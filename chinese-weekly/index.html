Grammar

1. Big numbers with 百/千/万 (hundred / thousand / ten-thousand)

Chinese counts in units of 十 (10), 百 (100), 千 (1,000), and 万 (10,000) — note that 万, not "hundred-thousand", is the next big unit after 千, so 100,000 is 十万 (ten wàn) and 10 千 = 1 万. When a place value is skipped inside the number, insert 零 (líng) once as a placeholder: 101 = 一百零一, 1001 = 一千零一, 209 = 二百零九. A trailing 十 can drop its 一: 110 is 一百一十 (not 一百十). Read every non-zero place with its unit. Structure: number + 百/千/万 (+ number + smaller unit); insert 零 for internal gaps.

五百八十🔊
wǔbǎi bāshí
580
五千零三十五🔊
wǔqiān líng sānshíwǔ
5,035
一百零一。🔊
Yìbǎi líng yī.
101.

2. 两 vs 二 before 百/千/万

When "two" comes before 百, 千, or 万 as a quantity, use 两: 两百 (200), 两千 (2,000), 两千七百 (2,700). Use 二 for the number two in counting and in the tens/ones and after 十 (e.g. 十二, 二十). So 2,000 is 两千, and 二千 is marked wrong in the lesson. Memory aid: 两 = "a pair/amount of," 二 = "the digit two." Structure: 两 + 百/千/万 (quantity); 二 elsewhere (counting, 十 place).

两千七百🔊
liǎngqiān qībǎi
2,700
一百二十🔊
yìbǎi èrshí
120
这个手表两千块。🔊
Zhège shǒubiǎo liǎngqiān kuài.
This watch is 2,000 kuai.

3. 打X折 — the discount system

打折 (dǎ zhé) literally is "apply a fold/fraction." The number is how many tenths you PAY, not the amount off — so 打七折 means you pay 70% (i.e. 30% off), 打八折 = pay 80% (20% off), 打八五折 = pay 85% (15% off). To ask the rate, use 打几折 (dǎ jǐ zhé, "what discount?"). 不打折 = no discount. This is the reverse of the English "X% off," so always convert: 打 X 折 → (10−X)0% off. Structure: 打 + number(s) + 折 (the number = tenths of price you PAY).

这件衣服一百元，现在打七折。🔊
Zhè jiàn yīfu yìbǎi yuán, xiànzài dǎ qī zhé.
This clothing is 100 yuan, now 30% off (pay 70).
请问这件衣服打几折？🔊
Qǐngwèn zhè jiàn yīfu dǎ jǐ zhé?
Excuse me, what's the discount on this?
这是最新的设计，不打折。🔊
Zhè shì zuìxīn de shèjì, bù dǎ zhé.
This is the newest design, no discount.

4. 有点儿 + Adjective vs Adjective + 一点儿

有点儿 goes BEFORE an adjective and usually expresses a mild complaint or something undesirable — 有点儿贵 = "(unfortunately) a bit expensive." 一点儿 comes AFTER the adjective and asks for or states a small degree of change — 便宜一点儿 = "(make it) a bit cheaper." Don't swap them: 便宜有点儿 and 有点儿便宜一点儿 are wrong. Memory aid: 有点儿 = "somewhat (and I'm not happy)"; Adj+一点儿 = "a little more ___, please." Structure: 有点儿 + Adj (complaint about current state); Adj + 一点儿 (request a change / comparison).

这件衬衫有点儿贵。🔊
Zhè jiàn chènshān yǒudiǎnr guì.
This shirt is a bit (too) expensive.
太贵了，便宜一点儿吧？🔊
Tài guì le, piányi yìdiǎnr ba?
Too expensive, make it a bit cheaper?
我懂一点儿中文。🔊
Wǒ dǒng yìdiǎnr Zhōngwén.
I understand a little Chinese.

5. 太 + Adjective + 了 (too / excessively)

太……了 brackets an adjective to mean "too / excessively ___," usually with an emotional charge (dismay, admiration). 太贵了 = "too expensive!" The 了 is part of the frame and normally can't be dropped. It differs from 很 (just "very"): 太贵了 implies "more than I'll accept," while 很贵 is a neutral "very expensive." Structure: 太 + Adjective + 了.

那件衣服两千七百元，太贵了。🔊
Nà jiàn yīfu liǎngqiān qībǎi yuán, tài guì le.
That clothing is 2,700 yuan, too expensive.
这个设计太新了！🔊
Zhège shèjì tài xīn le!
This design is so new!
太贵了！🔊
Tài guì le!
Too expensive!
