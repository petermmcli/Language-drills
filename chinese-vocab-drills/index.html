1. yellow

► Pinyin
A. 颜色 — yánsè
B. 黄色 — huángsè
C. 红色 — hóngsè
D. 大小 — dàxiǎo
► Answer
B. 黄色 (huángsè) = yellow. 颜色 = color in general; 红色 = red.

2. again / once more (future action)

► Pinyin
A. 又 — yòu
B. 还 — hái
C. 再 — zài
D. 短 — duǎn
► Answer
C. 再 (zài) = again for future actions. 又 (yòu) = again for past/completed actions. 再见 = goodbye (see you again).

3. shoes

► Pinyin
A. 衣服 — yīfu
B. 鞋 — xié
C. 颜色 — yánsè
D. 数学 — shùxué
► Answer
B. 鞋 (xié) = shoes. 衣服 = clothes. Combined: 鞋店 = shoe store.

4. to give an example

► Pinyin
A. 告诉 — gàosu
B. 例子 — lìzi
C. 举例子 — jǔ lìzi
D. 记得 — jìde
► Answer
C. 举例子 (jǔ lìzi) = to give an example. 举 = to raise/offer; 例子 = example. 告诉 = to tell.

5. to remember

► Pinyin
A. 记得 — jìde
B. 知道 — zhīdào
C. 明白 — míngbai
D. 忘 — wàng

Section 3: Fill in the Blank

Choose the word that best completes the sentence. Use 🔊 to hear the full sentence with the correct answer filled in.

1. 老师站在我______。 🔊

► Pinyin
Lǎoshī zhàn zài wǒ ______。
► Answer
B. 面前 — 老师站在我面前。
Lǎoshī zhàn zài wǒ miànqián。
The teacher is standing in front of me. — 面前 = in front of [someone]. 面 = face/front; 前 = forward/front. Compare: 后面 = behind, 旁边 = beside.

2. 请问______在哪儿？ 🔊

► Pinyin
Qǐngwèn ______ zài nǎr？
► Answer
C. 试衣间 — 请问试衣间在哪儿？
Qǐngwèn shìyījiān zài nǎr？
Excuse me, where is the fitting room? — 试衣间 = fitting room. 试 = to try on; 衣 = clothes; 间 = room. Don't confuse with 洗手间 (bathroom).

3. 我们去书______看看吧。 🔊

► Pinyin
Wǒmen qù shū ______ kànkàn ba。
► Answer
C. 店 — 我们去书店看看吧。
Wǒmen qù shūdiàn kànkàn ba。
Let's go take a look at the bookstore. — 店 (diàn) = store/shop. It compounds freely: 书店 = bookstore, 鞋店 = shoe store, 咖啡店 = coffee shop.

4. 我在______等你。 🔊

► Pinyin
Wǒ zài ______ děng nǐ。
► Answer
A. 地铁站 — 我在地铁站等你。
Wǒ zài dìtiězhàn děng nǐ。
I'm waiting for you at the subway station. — 地铁站 = subway station. 地铁 = subway; 站 = station/stop. Compare: 高铁站 = high-speed rail station.

5. 你可以______吗？ 🔊

► Pinyin
Nǐ kěyǐ ______ ma？
► Answer
C. 给我一个例子 — 你可以给我一个例子吗？
Nǐ kěyǐ gěi wǒ yī ge lìzi ma？
Can you give me an example? — 给我一个例子 = give me an example. Structure: 给 + person + 一个 + noun. Also: 举例子 uses the verb 举 instead of 给.
Section 4: English → Chinese Sentence Translation

Read the English sentence. Write the Chinese translation on paper or in your head, then toggle the answer. Use 🔊 next to the Chinese answer to hear it.

1. Translate: "This jacket is a bit short."

► Answer
这件衣服有点儿短。 🔊
Zhè jiàn yīfu yǒudiǎnr duǎn。
This jacket is a bit short.
短 (duǎn) = short (length). Opposite: 长 (cháng) = long. 有点儿 = a little bit.

2. Translate: "Last time I went to Shanghai."

► Answer
上次我去上海了。 🔊
Shàngcì wǒ qù Shànghǎi le。
Last time I went to Shanghai.
上次 = last time. 这次 = this time; 下次 = next time. 了 marks the action as completed.

3. Translate: "Do you like this color?"

► Answer
你喜欢这个颜色吗？ 🔊
Nǐ xǐhuān zhège yánsè ma？
Do you like this color?
颜色 (yánsè) = color. Common colors: 黄色 yellow, 红色 red, 白色 white. 有别的颜色的吗 = Do you have other colors?

4. Translate: "How is the size of this shirt?"

► Answer
这件衣服的大小怎么样？ 🔊
Zhè jiàn yīfu de dàxiǎo zěnmeyàng？
How is the size of this shirt?
大小 = size (literally big-small). Sizes: 大号 large, 中号 medium, 小号 small. 衣服的大小怎么样 is a natural way to ask about fit.

5. Translate: "It depends on the situation."

► Answer
看情况。 🔊
Kàn qíngkuàng。
It depends on the situation.
看情况 = it depends (literally 'look at the situation'). 看 = to see; 情况 = situation. A common conversational reply when something is uncertain.
